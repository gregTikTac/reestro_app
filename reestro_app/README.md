# ЕГРН-Парсер — десктоп-приложение

Графический интерфейс для парсера открытых сведений ЕГРН (Контур.Реестро).
Подробный план — в [ARCHITECTURE.md](ARCHITECTURE.md).

## Для конечного пользователя (заказчик)

**Подробная инструкция:** [ИНСТРУКЦИЯ.md](ИНСТРУКЦИЯ.md) — установка, все вкладки, сценарии работы, решение проблем.

Кратко:

1. Запустить `Setup.exe` (или распаковать папку `ЕГРН-Парсер`).
2. Открыть `ЕГРН-Парсер.exe`.
3. На вкладке **«Подключение»** ввести `apiKey`, `orgId` → «Проверить подключение».
4. На вкладке **«Пакетная обработка»** выбрать таблицу, папку, при необходимости
   диапазон объектов → «Старт».

Python и установка зависимостей вручную **не требуются** — всё внутри приложения.

## Для разработки

```bat
cd reestro_app
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python -m egrn_gui.main
```

## Сборка .exe

```bat
cd reestro_app
venv\Scripts\pyinstaller.exe build\egrn.spec --noconfirm --clean
```

Результат: `dist\ЕГРН-Парсер\ЕГРН-Парсер.exe` (папка ~125 МБ, всё внутри —
Python и зависимости конечному пользователю не нужны).

### Установщик (необязательно)

Открыть `build\installer.iss` в **Inno Setup Compiler** → Compile.
Получится `dist_installer\ЕГРН-Парсер_Setup_0.1.0.exe` с ярлыками и удалением.

Иконка приложения — `assets\icon.ico` (сгенерирована из `img.jpg`).
Подпись «by BeRealBear» — в правом нижнем углу окна.
