	Описание объекта закупки
на оказание услуг по предоставлению сведений о характеристиках объектов недвижимости на основании данных Единого государственного реестра недвижимости

1. Общие положения
1.1. Настоящее Техническое задание определяет объем, содержание, требования к качеству и срокам оказания услуг по подготовке информационных отчетов о характеристиках объектов недвижимости на основании сведений Единого государственного реестра недвижимости (далее - ЕГРН).
1.2. Услуги оказываются на основании сведений, размещенных в ЕГРН и доступных через официальный информационный ресурс Росреестра.Исполнитель обязан использовать актуальные сведения, доступные на дату формирования отчета.
1.3. Результатом оказания услуг является информационный отчет об объекте недвижимости.
1.4. Отчет носит справочный (информационный) характер и не является официальной выпиской из ЕГРН.

2. Порядок оказания услуг
Исполнитель обязуется по заданию Заказчика осуществлять поиск сведений об объектах недвижимости и формировать отчеты о характеристиках объектов недвижимости на основании сведений ЕГРН.
Отчет формируется по каждому объекту недвижимости отдельно.

3. Порядок взаимодействия при оказании услуг
3.1. Заказчик передает Исполнителю перечень объектов недвижимости.
3.2. Перечень объектов передается в электронном виде и должен содержать:
• адрес объекта недвижимости;
• кадастровый номер (при наличии).
3.3. Формат передачи данных:
• XLSX;
• CSV;
• либо иной формат, согласованный сторонами.
3.4. Каждый адрес считается отдельной единицей задания.

4. Требования к содержанию отчета
4.1 Сведения о характеристиках объекта:
• вид объекта недвижимости;
• кадастровый номер;
• адрес объекта;
• площадь;
• наименование объекта.
4.2 Сведения о зарегистрированных правах:
• вид зарегистрированного права;
• форма собственности;
• дата регистрации права;
• доля в праве собственности каждого из собственников.

5. Формат результата оказания услуг
5.1. Отчет предоставляется Заказчику в электронном виде.
5.2. Формат отчета: PDF.
5.3. Один объект недвижимости — один файл отчета.
5.4. Наименование файла: Адрес_объекта_или_Кадастровый_номер.pdf

6. SLA и сроки оказания услуг
6.1. Исполнитель обязан сформировать отчет по каждому адресу из полученного перечня, в срок не более 5 рабочих дней с момента получения задания.
6.2. Время отсчета начинается с момента получения Исполнителем списка адресов (перечня).
6.3. В случае невозможности получения сведений по объекту недвижимости Исполнитель обязан указать это в отчете.

7. Требования к качеству услуг
Исполнитель обязан:
• использовать актуальные сведения ЕГРН;
• обеспечивать корректность переноса данных;
• не изменять содержание полученных сведений.

Ошибкой считается:
• несоответствие данных ЕГРН;
• неполный перенос данных;
• неверная идентификация объекта.

8. Отчетность Исполнителя
По итогам календарного месяца Исполнитель обязан предоставить Заказчику отчет о выполненных работах.
Отчет должен содержать:
• общее количество полученных адресов;
• количество сформированных отчетов;
• количество адресов, по которым сведения отсутствуют.
